# UIUX Brief — Toyota Sales Representative Website (Malaysia)

## Project goal
Refresh and strengthen the existing Toyota sales representative landing page so it feels more trustworthy, clearer, and more conversion-focused for Malaysia users.

## Current live site
- Live URL: https://miniteatree.github.io/toyota-sales-malaysia/
- Current format: static single-page landing page
- Primary CTA: WhatsApp enquiry

## Working assumptions for this round
- This is a refinement / redesign pass on the current website, not a brand-new product
- Keep static-site delivery unless a later phase proves otherwise
- Mobile-first remains critical
- Keep WhatsApp as the main conversion path
- Use safer, more honest copy: avoid fake urgency, fake promo specifics, or unsupported claims

## Existing gaps observed
- Placeholder WhatsApp number still appears in multiple CTA links
- Placeholder branch / dealership details reduce trust
- Placeholder testimonials weaken credibility
- Current content is broad but a bit generic; value proposition is not sharp enough
- Visual hierarchy can be improved so users know what to do faster
- FAQ / trust / process sections can be more concise and more local
- Current structure does not fully reflect the broader multi-model focus in the implementation history

## Target audience
- Penang and nearby Malaysia buyers researching Toyota options
- Users comparing Toyota categories before contacting a sales advisor
- Mobile visitors from WhatsApp, Facebook, TikTok, Instagram, and search
- Users who want a quick quotation / financing estimate / booking guidance

## Primary conversion action
1. WhatsApp Lim directly for model enquiry / quotation / test drive / financing guidance

## Secondary trust goals
- Make Lim feel legitimate, responsive, and helpful
- Reduce hesitation caused by generic sales copy
- Clarify what happens after a WhatsApp message
- Make it easy to choose a model category and start an enquiry

## Must-have sections for next UIUX pass
- Hero with clear local trust signal and strong WhatsApp CTA
- Short credibility / trust strip
- About Lim / advisor support section
- Toyota model lineup or grouped categories
- Financing / quotation / process explanation
- Why buyers work with Lim
- FAQ
- Branch / location / Penang coverage
- Sticky WhatsApp CTA

## Nice-to-have improvements
- Better model grouping and scanability
- Better CTA copy per section
- Real-world proof placeholders with a clearer upgrade path for later replacement
- More premium, cleaner visual system while staying practical

## Constraints
- Static frontend for now
- No backend / CMS required in this phase
- Do not publish inaccurate pricing or promo claims
- Keep implementation maintainable and GitHub Pages-friendly

## Success criteria
- Stronger first impression within 5 seconds
- Clearer trust and process explanation
- Higher CTA clarity on mobile
- Cleaner visual hierarchy and more polished layout
- Ready for a later content pass once real WhatsApp / branch / testimonial details are available
